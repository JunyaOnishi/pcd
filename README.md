# pcd
peco cd for windows with nyagos

Needs
-------

* [peco](https://github.com/peco/peco)
* [nyagos](https://github.com/zetamatta/nyagos)
* Windows

How to Install
-------
copy pcd.lua to nyagos.d directory.
